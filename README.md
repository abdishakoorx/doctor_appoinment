# DoctorWhere 👨‍⚕️

**Comprehensive Doctor Appointment Management System**

[Live Demo](https://doctorwhere.vercel.app/)

## Overview

DoctorWhere bridges the gap between patients and healthcare providers with a streamlined appointment booking system. Featuring real-time availability, doctor profiles, and seamless appointment management for modern healthcare needs.

## ✨ Key Features

- **Smart Appointment Booking** - Real-time doctor availability and scheduling
- **Doctor Directory** - Comprehensive profiles with specializations and ratings
- **Patient Dashboard** - Manage appointments, medical history, and preferences
- **Secure Authentication** - HIPAA-compliant user management with Kinde
- **GraphQL Integration** - Efficient data fetching and real-time updates
- **Mobile-First Design** - Optimized for all devices and screen sizes

## 🛠️ Tech Stack

- **Frontend:** Next.js, React, Tailwind CSS
- **UI Components:** shadcn/ui
- **Authentication:** Kinde Auth
- **API Layer:** GraphQL
- **Content Management:** Hygraph CMS
- **Deployment:** Vercel
